# Event Taxonomy

This document defines all user events tracked in the FinPath system.
Events represent meaningful user actions across the fintech product lifecycle.

## 1. Acquisition & Entry

- app_opened  
- landing_page_viewed  
- signup_started  
- signup_completed  
- email_verified  
- login  
- logout  


## 2. Onboarding

- onboarding_started  
- onboarding_completed  
- profile_completed  
- investment_goal_selected  


## 3. Risk Profiling

- risk_quiz_started  
- risk_question_answered  
- risk_quiz_completed  
- risk_score_generated  
- risk_profile_updated  


## 4. KYC

- kyc_started  
- kyc_document_uploaded  
- kyc_verification_failed  
- kyc_completed  


## 5. Funding

- bank_account_linked  
- deposit_initiated  
- deposit_completed  
- deposit_failed  
- withdrawal_initiated  
- withdrawal_completed  


## 6. Portfolio Lifecycle

- portfolio_created  
- portfolio_viewed  
- portfolio_rebalanced  
- portfolio_allocation_changed  
- portfolio_closed  


## 7. Investment Actions

- investment_added  
- investment_removed  
- auto_invest_enabled  
- auto_invest_disabled  
- dividend_received  


## 8. Feature Usage

- goal_created  
- goal_updated  
- goal_deleted  
- notification_clicked  
- feature_viewed  


## 9. Engagement

- dashboard_viewed  
- performance_chart_viewed  
- portfolio_report_downloaded  
- support_chat_opened  


## 10. Retention & Exit

- account_inactive  
- withdrawal_all_funds  
- account_closed
