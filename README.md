# FinPath

**End-to-end real-time onboarding & activation analytics platform for a robo-advisory fintech application**

## Overview

Vest Activation Funnel is a complete, production-like analytics system that tracks and analyzes the full user onboarding journey in a modern robo-advisory platform (think Wealthfront / Betterment clone).

It ingests synthetic user events in real time, stores them durably, computes critical product metrics (funnels, drop-offs, time-to-activate, feature adoption, cohort retention), exposes them via a clean API, and visualizes everything in a live dashboard.

The primary goal is to surface actionable insights about friction points — especially around KYC, bank linking, and first deposit — so a product/growth team could prioritize fixes and measure real uplift.

**Key focus areas:**
- Onboarding funnel drop-off analysis
- Time-to-activation & behavioral segmentation
- Feature adoption (auto-rebalance, goal setting)
- Real-time streaming ingestion + analytical SQL layer
